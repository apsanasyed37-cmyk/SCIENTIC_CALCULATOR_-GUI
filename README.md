# 🧮 Scientific Calculator

A Scientific Calculator developed using **Python** and **Tkinter**. This application provides basic arithmetic operations along with scientific functions through a user-friendly graphical interface.

---

## 📌 Features

- ➕ Addition
- ➖ Subtraction
- ✖ Multiplication
- ➗ Division
- 🔢 Decimal calculations
- √ Square Root
- 📐 Trigonometric Functions
  - sin()
  - cos()
  - tan()
- 📊 Logarithm (Base 10)
- π (Pi) Constant
- Exponentiation (^)
- Parentheses Support
- Clear (C) Button
- Error Handling for invalid expressions

---

## 🛠 Technologies Used

- Python 3
- Tkinter (GUI)
- Math Module

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Scientific-Calculator.git
```

2. Open the project folder

```bash
cd Scientific-Calculator
```

3. Run the program

```bash
python scientific_calculator.py
```

---

## 💻 Requirements

- Python 3.x

No external libraries are required because **Tkinter** and **Math** are included with Python.

---

## 🎮 How to Use

- Enter numbers using the calculator buttons.
- Select the desired arithmetic or scientific operation.
- Press **=** to calculate the result.
- Press **C** to clear the display.
